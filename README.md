Naval Fighter
==========================

Éste es el README de la aplicación Naval Fighter.

Es una Aplicación web que alberga un juego, inspiriado en el mítico juego arcade 1942.

Herramientas utilizadas:
+ HTML 5
+ CSS 3
+ JavaScript
+ jQuery
+ Phaser
+ Bootstrap
+ JSON
